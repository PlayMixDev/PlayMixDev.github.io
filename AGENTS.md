# AGENTS.md

## О проекте
- Это простой статический сайт на GitHub Pages.
- Назначение сайта: показывать игры в Google Play, держать `app-ads.txt` и страницы privacy policy для игр.
- Основной стек: Jekyll templates + статические HTML/CSS/assets.

## Ключевые файлы
- `index.html` — главная страница со списком игр.
- `_layouts/default.html` — базовый layout.
- `_includes/game.html` — шаблон карточки игры.
- `_data/games.yml` — источник списка игр на главной.
- `css/styles.css` — стили сайта.
- `app-ads.txt` — файл для рекламных сетей.
- `PrivacyPolicy/` — markdown-файлы политик конфиденциальности.

## Как вносить изменения
- Если нужно добавить или изменить игру на главной, сначала правь `_data/games.yml`.
- Если нужно менять внешний вид карточек, правь `_includes/game.html` и `css/styles.css`.
- Если нужно обновить privacy policy, правь соответствующий файл в `PrivacyPolicy/`.
- Если нужно обновить рекламные идентификаторы или сети, правь только `app-ads.txt`.

## Правила
- Не усложняй проект без явной причины: это должен оставаться простой статический сайт.
- Не добавляй сборочные инструменты, frontend framework или лишний JS без прямого запроса.
- Не переименовывай существующие публичные файлы и пути без необходимости: GitHub Pages, privacy links и `app-ads.txt` должны оставаться доступными по стабильным URL.
- Не меняй картинки, JSON и переводы, если задача не требует этого прямо.
- Новые privacy policy добавляй отдельными файлами в `PrivacyPolicy/`.

## Проверка перед завершением
- Убедись, что ссылки на иконки и Google Play валидны.
- Убедись, что layout не сломан и список игр продолжает рендериться из `_data/games.yml`.
- Если менялся `app-ads.txt`, проверь формат строк и отсутствие случайных правок вне нужных записей.
